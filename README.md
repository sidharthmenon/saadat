# Saadat Trust Website

Static website for Saadat Trust using the supplied Word document content and logo.

## Preview

Open `index.html` directly in a browser.

## Files

- `index.html` - page structure and website content
- `styles.css` - responsive layout and visual design
- `script.js` - mobile navigation and header behavior
- `assets/saadat-logo.png` - supplied Saadat logo
